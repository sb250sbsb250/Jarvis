## 🎯 核心能力

### 1️⃣ 自主决策与执行
- 理解用户意图后**自主规划执行步骤**，无需逐条指令
- 一次只做一件事，步步验证，确认结果后再继续
- 失败时自动分析原因，换方案重试（同方式最多3次）
- 复杂任务完成后主动汇报进度和结论

### 2️⃣ 工具调用与系统集成
- **文件系统**：读写、编辑、搜索、批量处理
- **Excel 操作**：读取/写入/批量处理，支持保留格式
- **代码操作**：搜索、读取、编辑、diff 预览、回滚
- **Shell 执行**：运行脚本、自动化任务
- **网络访问**：网页抓取、搜索引擎查询
- **Git 操作**：status → commit → push 标准流程
- **Office 文档**：PDF/Word 读取、Word 写入
- **图片处理**：图片读取、OCR 文字识别
- **系统信息**：OS/CPU/内存/磁盘/时间

### 3️⃣ 数据处理与分析
- Excel/CSV 批量数据处理（Python 脚本一次性完成）
- 大文件分片读取、智能跳过空列
- 数据转换、填充、计算、验证
- 结果输出到文件，支持多种格式

### 4️⃣ 记忆与连续性
- 工作记忆：会话内记住已读取的数据，不重复读取
- 长期记忆：通过 MEMORY.md 和每日日志保持跨会话连续性
- 心跳机制：定时检查邮件、日历、通知，主动汇报

### 5️⃣ 代码开发与维护
- 代码搜索、阅读、理解、修改
- 自动备份 + diff 预览 + 回滚
- 多文件修改按依赖顺序处理
- 语法检查和逻辑完整性确认

## 🏆 核心优势

| 优势 | 说明 |
|------|------|
| **自主性** | 不需要用户确认即可执行搜索、读取、分析等操作 |
| **效率优先** | 能一步完成不分两步，能并行不串行，能脚本不逐次 |
| **容错能力** | 失败分析原因换方案，同方式最多3次，3种方式都失败才放弃 |
| **记忆复用** | 已获取的数据牢记于心，绝不重复读取同一内容 |
| **诚实透明** | 任务完成给出结论，无法完成说明原因和替代方案 |
| **安全可靠** | 不泄露隐私数据，不执行破坏性操作，重要文件先备份 |
